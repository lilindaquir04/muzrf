# МузРФ — Портал отечественной музыки

Учебный проект по дисциплине «Веб-разработка». Лабораторная работа №2.  
Тема: **Отечественная музыка**.

## Страницы сайта

| Файл | Описание |
|------|----------|
| `index.html` | Главная страница (Hero, статистика, популярные исполнители) |
| `artists.html` | Список исполнителей (таблица, поиск, фильтрация по жанру) |
| `add-artist.html` | Форма добавления исполнителя |
| `artist.html` | Страница конкретного исполнителя (Кино) |

## Технологии

- **Bootstrap 5.3** — сетка, компоненты, утилиты
- **Bootstrap Icons 1.11** — иконки
- **Google Fonts** — Playfair Display, Lora
- **Vanilla JS** — фильтрация таблицы, валидация формы
- Собственный файл стилей: `css/style.css`

## Структура проекта

```
muzrf/
├── index.html          # Главная страница
├── artists.html        # Список исполнителей
├── add-artist.html     # Форма добавления
├── artist.html         # Страница исполнителя
├── css/
│   └── style.css       # Собственные стили
└── img/
    ├── logo.svg        # Логотип сайта
    └── favicon.svg     # Favicon
```

## Запуск

Открыть `index.html` в браузере или опубликовать через GitHub Pages.
